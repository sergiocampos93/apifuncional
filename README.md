# API Funcional

Projeto experimental de **API REST em ASP.NET Core**, explorando práticas de desenvolvimento backend com **.NET**, **Entity Framework Core** e **ASP.NET Core Identity**.

## Tecnologias

* ASP.NET Core
* Entity Framework Core
* SQL Server
* ASP.NET Core Identity
* Swagger / OpenAPI

## Funcionalidades atuais

* CRUD de produtos
* Validações com Data Annotations
* Migrations com EF Core
* Integração com Identity

## Próximos passos

* Autenticação com JWT
* Autorização por roles
* DTOs para requests/responses
* Melhor organização em camadas
