# API Funcional

Projeto experimental de **API REST em ASP.NET Core**, explorando práticas de desenvolvimento backend com **.NET**, **Entity Framework Core** e **ASP.NET Core Identity**.

## Tecnologias

- ASP.NET Core
- Entity Framework Core
- SQL Server
- ASP.NET Core Identity
- JWT Authentication
- Role-based Authorization
- Swagger / OpenAPI

## Funcionalidades

- API REST para gerenciamento de produtos
- Autenticação com JWT
- Autorização baseada em roles
- Integração com ASP.NET Core Identity
- Persistência com Entity Framework Core
- Migrations para controle de schema do banco
- Documentação automática com Swagger
- Configuração de CORS para integração com aplicações frontend

## Próximos passos

- Paginação e filtros nos endpoints
- Melhor tratamento de erros
- Logs estruturados
- Testes unitários
